# wercker-php-symfony2-npm
Dockerfile for PHP/Symfony2 development using the Wercker EWOK stack.

Contains
- PHP5.6
- all extensions required to run symfony2
- configured php.ini with locale
- phpunit
- nodejs
- ruby / bundler for capifony deployments
